Auto Email Sender

The Auto Email Sender is a code written in Python that reads a .csv file which is a list of emails and send an email, written by you, to each person on that list. This code came to be when I was working as an intern where a job of mine was to send emails out to websites to see if they would advertise our service. This process was extremely tedious so I decided to create a code to cut down the amount of time spent significantly doing this task. (Made using Python IDLE)

Similar to the Auto Email Bot, the Clipboard Copier was also made as a tool to simplify my job at an internship. This code was written in Python and would create new hotkeys on my keyboard allowing me to make it so that when I clicked a certain key it would copy a prompt to my keyboard. This came in handy when mass recruiting on Indeed as I could copy a prompt to my clipboard depending on the application I got and paste it to send the applicant a message. (Made using PyCharm)
